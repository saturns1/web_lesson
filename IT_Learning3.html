Anaconda

Anaconda is a free and open-source system installer for Linux distributions. Anaconda is used by Red Hat Enterprise Linux, Oracle Linux, Scientific Linux, CentOS, Qubes OS, Fedora, Sabayon Linux and BLAG Linux and GNU, also in some less known and discontinued distros like Progeny Componentized Linux (based on Debian, founded by Ian Murdock), Asianux, Foresight Linux, Rpath Linux and VidaLinux (based on Gentoo Linux).
